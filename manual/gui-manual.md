### 主要区域说明

#### 工具栏

位于界面顶部，包含以下功能控件：

- **排序下拉框**：选择电影排序方式（名称、年份、评分、添加时间）
- **分类筛选**：按分类快速过滤电影
- **标签筛选**：按标签过滤电影
- **演员筛选**：按演员过滤电影
- **搜索框**：输入关键词搜索电影
- **视图切换**：切换网格/列表视图
- **刷新按钮**：手动刷新影库
- **设置按钮**：打开设置面板

#### 侧边栏

位于界面左侧，包含：

- **分类列表**：显示所有电影分类及电影数量
- **电影盒子列表**：显示所有电影盒子
- **添加盒子按钮**：快速创建新盒子

#### 电影展示区域

占据界面主要空间，以网格或列表形式展示电影海报。

#### 状态栏

位于界面底部，显示：

- 电影总数
- 演员总数
- 分类总数

## 电影展示

### 视图模式

电影支持两种视图模式：

#### 网格视图

以海报网格形式展示电影，视觉效果直观。

**操作步骤**：
1. 点击工具栏的"网格视图"图标
2. 电影以海报卡片形式排列
3. 每张卡片显示：海报、电影名称、年份、评分

【网格视图截图】

#### 列表视图

以列表形式展示电影，信息密度更高。

**操作步骤**：
1. 点击工具栏的"列表视图"图标
2. 电影以列表行形式排列
3. 每行显示：海报、电影名称、分类、年份、评分、标签

【列表视图截图】

### 排序功能

支持多种排序方式：

**操作步骤**：
1. 点击工具栏的"排序"下拉框
2. 选择排序方式：
   - **名称升序/降序**：按电影名称字母顺序
   - **年份升序/降序**：按发布年份
   - **评分升序/降序**：按评分高低
   - **添加时间升序/降序**：按添加到影库的时间

### 筛选功能

#### 按分类筛选

**操作步骤**：
1. 点击工具栏的"分类"下拉框
2. 选择目标分类（如"电影"、"电视剧"、"纪录片"、"动漫"）
3. 电影展示区域立即更新，仅显示该分类的电影
4. 或在侧边栏点击对应分类

#### 按标签筛选

**操作步骤**：
1. 点击工具栏的"标签"下拉框
2. 选择目标标签（如"动作"、"科幻"、"喜剧"）
3. 电影展示区域更新，仅显示包含该标签的电影

#### 按演员筛选

**操作步骤**：
1. 点击工具栏的"演员"按钮
2. 在弹出的演员选择器中搜索或浏览演员
3. 选择一个或多个演员
4. 点击"确认"按钮
5. 电影展示区域更新，仅显示包含所选演员的电影

#### 按新电影筛选

**操作步骤**：
1. 勾选工具栏的"仅显示新电影"复选框
2. 系统根据设置中的"新电影判定时间"（默认168小时/7天）筛选
3. 仅显示最近添加的电影

#### 搜索电影

**操作步骤**：
1. 在搜索框中输入关键词
2. 点击搜索按钮或按回车键
3. 系统匹配电影名称、导演、演员、简介等字段
4. 显示匹配结果
5. 点击"清除"按钮清空搜索

## 电影管理

### 添加电影

支持手动添加和批量导入两种方式。

#### 手动添加单部电影

**操作步骤**：
1. 点击工具栏的"添加电影"按钮
2. 弹出"添加电影"对话框
3. 填写电影信息：
   - **名称**（必填）：电影名称
   - **分类**（必填）：选择电影分类
   - **年份**：发布年份
   - **导演**：导演姓名
   - **演员**：点击"选择演员"按钮，从演员库中选择
   - **标签**：点击"选择标签"按钮，从标签库中选择
   - **评分**：0-10分的评分
   - **时长**：电影时长（分钟）
   - **简介**：电影剧情简介
4. 点击"保存"按钮
5. 系统生成电影ID（如 movie-001）并创建 NFO 文件

【添加电影对话框截图】

#### 从网络搜索添加（TMDB/R18）

**操作步骤**：
1. 点击"添加电影"对话框中的"搜索"按钮
2. 在弹出的搜索框中输入电影名称
3. 选择数据源（TMDB 或 R18）
4. 点击"搜索"按钮
5. 从搜索结果中选择目标电影
6. 系统自动填充电影信息
7. 可进一步编辑补充信息
8. 点击"保存"按钮

【网络搜索电影截图】

#### 批量导入

**操作步骤**：
1. 点击菜单栏"电影" → "批量导入"
2. 选择导入方式：
   - **从目录导入**：选择包含电影 NFO 文件的目录
   - **从JSON文件导入**：选择 JSON 格式的导入文件
3. 系统扫描并显示待导入电影列表
4. 勾选需要导入的电影
5. 点击"开始导入"按钮
6. 显示导入结果统计

【批量导入对话框截图】

### 编辑电影

#### 查看电影详情

**操作步骤**：
1. 在电影展示区域点击目标电影的海报或卡片
2. 弹出"电影详情"窗口
3. 显示完整电影信息：
   - 海报（大图）
   - 名称、分类、年份
   - 导演、演员列表
   - 标签列表
   - 评分、时长
   - 简介
   - 文件列表（视频文件）

【电影详情窗口截图】

#### 编辑电影信息

**操作步骤**：
1. 在电影详情窗口点击"编辑"按钮
2. 进入编辑模式，所有字段变为可编辑状态
3. 修改需要更新的字段
4. 点击"保存"按钮保存修改
5. 或点击"取消"按钮放弃修改

【编辑电影截图】

#### 管理电影文件

**操作步骤**：
1. 在电影详情窗口切换到"文件管理"标签页
2. 查看当前关联的视频文件列表
3. **添加文件**：
   - 点击"添加文件"按钮
   - 选择本地视频文件
   - 文件添加到列表
4. **移除文件**：
   - 选择目标文件
   - 点击"移除"按钮
   - 确认删除
5. **设置主文件**：
   - 选择目标文件
   - 点击"设为主文件"按钮
   - 该文件将作为默认播放文件

【文件管理截图】

#### 管理海报

**操作步骤**：
1. 在电影详情窗口的海报区域
2. **上传本地海报**：
   - 点击海报区域的"上传"图标
   - 选择本地图片文件
   - 海报立即更新
3. **从网络下载海报**：
   - 点击"搜索海报"按钮
   - 系统从 TMDB/R18 搜索海报
   - 选择满意的海报
   - 点击"下载并应用"

### 删除电影

**操作步骤**：
1. 在电影详情窗口点击"删除"按钮
2. 弹出确认对话框："确认删除电影 XXX？"
3. 点击"确认"按钮
4. 系统删除 NFO 文件和相关数据
5. 电影从展示列表中移除

### 批量操作

#### 批量选择电影

**操作步骤**：
1. 在网格视图中，鼠标悬停在电影卡片上
2. 点击卡片右上角的复选框
3. 或直接拖拽鼠标框选多个电影
4. 选中的电影显示高亮边框

#### 批量删除

**操作步骤**：
1. 选择多个电影（见上方"批量选择电影"）
2. 点击工具栏的"批量删除"按钮
3. 弹出确认对话框，显示待删除电影数量
4. 点击"确认"按钮
5. 系统批量删除选中的电影

## 电影播放

### 打开播放器

**操作步骤**：
1. 在电影详情窗口的"文件管理"标签页
2. 选择要播放的视频文件
3. 点击"播放"按钮
4. 或在电影列表中右键点击电影，选择"播放"
5. 系统打开播放器窗口

【播放器窗口截图】

### 播放器界面

播放器包含以下控件：

```
┌────────────────────────────────────────────┐
│                                            │
│                                            │
│             视频显示区域                     │
│                                            │
│                                            │
├────────────────────────────────────────────┤
│ [播放/暂停] [上一集] [下一集] [进度条]      │
│ [音量] [全屏] [播放列表] [设置]             │
└────────────────────────────────────────────┘
```

### 播放控制

#### 基本控制

- **播放/暂停**：点击播放按钮或按空格键/K键
- **停止**：点击停止按钮
- **上一集**：点击上一集按钮（如果有播放列表）
- **下一集**：点击下一集按钮（如果有播放列表）

#### 进度控制

- **进度条**：拖拽进度条跳转到指定位置
- **快进**：按右键（快进10秒）
- **快退**：按左键（快退10秒）
- **跳转**：在进度条上点击目标位置

#### 音量控制

- **音量滑块**：拖拽调整音量
- **静音**：点击音量图标或按M键
- **音量增加**：按上键
- **音量减少**：按下键

#### 全屏控制

- **全屏**：点击全屏按钮或按F键
- **退出全屏**：按ESC键

### 播放列表

**操作步骤**：
1. 在播放器窗口点击"播放列表"按钮
2. 右侧弹出播放列表面板
3. 显示当前播放列表中的所有视频
4. 点击列表中的任一视频切换播放
5. 支持拖拽调整播放顺序
6. 播放完当前视频后自动播放下一个

### 播放进度记忆

系统自动保存播放进度：

- 暂停或关闭播放器时，自动保存当前播放位置
- 下次播放同一文件时，自动跳转到上次位置
- 在进度条上显示上次播放位置标记

## 演员管理

### 打开演员管理界面

**操作步骤**：
1. 点击菜单栏"管理" → "演员管理"

   - **姓名**（必填）：演员姓名

   - **生日**：出生日期
   - **简介**：演员简介

### 编辑演员

**操作步骤**：
1. 在演员列表中点击目标演员
2. 右侧显示演员详细信息
3. 点击"编辑"按钮
4. 修改演员信息
5. 点击"保存"按钮

### 删除演员

**操作步骤**：
1. 在演员列表中选中目标演员
2. 点击"删除"按钮
3. 弹出确认对话框
4. 点击"确认"按钮
5. 演员从列表中移除

1. 在演员列表中点击目标演员
2. 右侧显示演员详情
3. 切换到"参演电影"标签页
4. 显示该演员参演的所有电影列表
5. 点击任一电影可跳转到电影详情

## 电影盒子管理

### 什么是电影盒子

电影盒子（MovieBox）是自定义电影合集，类似于播放列表。可以按主题、状态、个人喜好等维度组织电影。

每个盒子中的电影可以设置：
- **观看状态**：未观看（unwatched）、观看中（watching）、已观看（watched）
- **评分**：0-10分的个人评分

### 创建盒子

**操作步骤**：
1. 点击侧边栏的"添加盒子"按钮（+图标）
2. 弹出"创建盒子"对话框
3. 填写盒子信息：
   - **盒子名称**（必填）：如"科幻经典"、"我的待看"
   - **描述**：盒子描述（可选）

## 设置功能

### 打开设置面板

**操作步骤**：
1. 点击工具栏的"设置"按钮（齿轮图标）
2. 或点击菜单栏"文件" → "设置"
3. 弹出设置对话框

【设置面板截图】

### 设置标签页

设置对话框包含4个标签页：

#### 1. 基本设置 (basic-settings)

**主题设置**：
- **主题**：选择 dark（暗色）或 light（亮色）
- **语言**：选择 zh-CN（中文）或 en-US（英文）

**界面设置**：
- **侧边栏宽度**：输入像素值（默认200）
- **海报尺寸**：选择 small/medium/large
- **海报样式**：选择 cover（裁剪填充）或 contain（完整显示）
- **网格列数**：输入列数（默认6）
- **视图模式**：选择 grid（网格）或 list（列表）

**操作步骤**：
1. 切换到"基本设置"标签页
2. 修改各项设置
3. 点击"保存"按钮应用更改

【基本设置截图】

#### 2. 路径设置 (path-settings)

配置各种文件路径：

**电影库路径**：
- **电影目录**（moviesDir）：存放按分类组织的电影文件夹
  - 点击"选择目录"按钮浏览选择
  - 或直接输入路径
- **电影盒子目录**（movieboxDir）：存放盒子定义文件
- **演员照片目录**（actorPhotoDir）：存放演员头像图片

**视频解析路径**：
- **ffmpeg路径**：ffmpeg 可执行文件路径
  - 点击"选择文件"按钮浏览选择
- **ffprobe路径**：ffprobe 可执行文件路径

**操作步骤**：
1. 切换到"路径设置"标签页
2. 填写或选择各路径
3. 点击"保存"按钮

【路径设置截图】

#### 3. 电影设置 (movie-settings)

**新电影判定**：
- **新电影小时数**：定义多少小时内添加的电影视为"新电影"（默认168小时=7天）

**TMDB配置**：
- **API地址**：TMDB API 基础URL（默认 https://api.themoviedb.org）
- **语言**：返回数据的语言（默认 zh-CN）
- **API Token**：TMDB API 访问令牌（必需，从 TMDB 网站获取）

**R18数据库配置**：
- **数据库URL**：PostgreSQL 连接字符串
  - 格式：`postgresql://host:port/database`
- **用户名**：数据库用户名
- **密码**：数据库密码

**操作步骤**：
1. 切换到"电影设置"标签页
2. 填写各项配置
3. 点击"测试连接"按钮验证配置（TMDB/R18）
4. 点击"保存"按钮

【电影设置截图】

#### 4. 网络设置 (network-settings)

**代理配置**：
- **启用代理**：勾选以启用HTTP/HTTPS代理
- **代理地址**：代理服务器地址和端口
  - 格式：`http://proxy.example.com:8080`
- **用户名**：代理认证用户名（可选）
- **密码**：代理认证密码（可选）

**操作步骤**：
1. 切换到"网络设置"标签页
2. 勾选"启用代理"
3. 填写代理地址和认证信息
4. 点击"测试代理"按钮验证连接
5. 点击"保存"按钮

【网络设置截图】

### 保存和取消设置

**保存设置**：
1. 修改完所有需要的设置后
2. 点击底部的"保存"按钮
3. 设置写入 `config/settings.json` 文件
4. 部分设置立即生效（如主题切换）
5. 部分设置需要重启应用生效（如路径变更）

**取消修改**：
1. 点击底部的"取消"按钮
2. 放弃所有未保存的修改
3. 设置对话框关闭

### 重置设置

**操作步骤**：
1. 在设置对话框中点击"重置"按钮
2. 弹出确认对话框："确认重置所有设置为默认值？"
3. 点击"确认"按钮
4. 所有设置恢复为出厂默认值

## 常见问题

### Q1: 如何导入已有的电影库？

**A**: 使用批量导入功能：
1. 确保每部电影有标准的 NFO 文件
2. 点击菜单栏"电影" → "批量导入"
3. 选择包含电影的目录
4. 系统自动扫描并导入

### Q2: TMDB 搜索返回空结果？

**A**: 检查以下设置：
1. 打开设置 → 电影设置
2. 确认 TMDB Token 已正确填写
3. 确认网络连接正常
4. 如果使用代理，检查代理设置

### Q3: 海报无法显示？

**A**: 可能原因：
1. 海报文件不存在：重新上传或下载海报
2. 路径配置错误：检查 settings.json 中的路径配置
3. 文件权限问题：确保应用有读取权限

### Q4: 播放器无法播放某些格式？

**A**: 内置播放器基于 HTML5 Video，支持常见格式：
- ✅ MP4 (H.264)
- ✅ WebM
- ✅ OGG
- ❌ MKV (可能需要转换)
- ❌ AVI (可能需要转换)

建议使用 MP4 (H.264编码) 格式。

### Q5: 如何备份电影数据？

**A**: 备份以下目录和文件：
1. `movies/` 目录（包含所有 NFO 文件）
2. `boxes/` 目录（盒子定义）
3. `actor/` 目录（演员照片）
4. `config/` 目录（配置文件）

将这些目录复制到安全位置即可。
