# 工作日志 - 电影字幕配置功能

## 目标

为电影管理程序添加播放器字幕配置功能，允许用户自定义字幕背景颜色和字体大小。

## 工作过程

### 第一步：探索代码库
通过 Read 工具阅读以下文件了解现有结构：
- `config/settings.json` - settings.json 结构，包含 proxy 等配置
- `src/main/services/HardCodeService.js` - getDefaultSettings() 方法返回默认配置
- `src/main/services/SettingsService.js` - 各 getter/setter 方法实现（如 setProxyConfig）
- `src/renderer/index.html` - settings modal 的 tab 结构和布局
- `src/renderer/js/main.js` - 元素引用、loadSettings()、事件监听、saveSettingsHandler()
- `src/renderer/js/player.js` - 字幕加载和显示逻辑
- `test/svc/SettingsService.test.js` - 现有测试结构和风格

### 第二步：实现

#### 2.1 修改 HardCodeService.js - 添加默认播放器配置

**文件**: `src/main/services/HardCodeService.js`

在 `getDefaultSettings()` 返回对象中，在 `proxy` 之后添加：
```javascript
player: {
    subtitle: {
        backgroundColor: 'rgba(0, 0, 0, 0.7)',
        fontSize: '22px'
    }
}
```

#### 2.2 修改 SettingsService.js - 添加 getter/setter 方法

**文件**: `src/main/services/SettingsService.js`

添加 `getPlayerConfig()` 和 `setPlayerConfig()` 方法。注意 `setPlayerConfig` 需要深度合并 subtitle 对象，以保留其他字段：
```javascript
setPlayerConfig(config) {
    if (!this.settings.player) {
        this.settings.player = {};
    }
    if (config.subtitle) {
        this.settings.player.subtitle = { ...this.settings.player.subtitle, ...config.subtitle };
    } else {
        this.settings.player = { ...this.settings.player, ...config };
    }
    this.saveSettings(this.settings);
}
```

#### 2.3 修改 config/settings.json - 添加 player 配置

**文件**: `config/settings.json`

在 `proxy` 配置之后添加 player 配置。

#### 2.4 修改 index.html - 添加播放器配置 Tab 页

**文件**: `src/renderer/index.html`

- Tab 按钮：`<button class="tab-btn" data-tab="player-settings">播放器配置</button>`
- Tab 内容：包含字幕背景配置（透明/底色单选）、背景颜色选择器、字体大小下拉框

#### 2.5 修改 main.js

**文件**: `src/renderer/js/main.js`

- **元素引用**：添加 subtitleBgTransparent、subtitleBgColored、subtitleBgColor、subtitleBgColorPreview、subtitleColorItem、subtitleFontSize
- **loadSettings()**：解析 rgba 颜色值转换为 hex 显示，加载背景配置和字体大小
- **事件监听**：字幕背景透明/底色切换、颜色选择器实时预览
- **saveSettingsHandler()**：添加 player.subtitle 配置保存

#### 2.6 修改 player.js - 应用动态字幕样式

**文件**: `src/renderer/js/player.js`

- 添加 `applySubtitleSettings()` 函数：从 settings 读取字幕配置应用到 subtitleDisplay 元素
- 在 DOMContentLoaded 初始化时调用
- 在 `autoLoadSubtitle()` 字幕加载成功后调用
- 在 `handleSubtitleSwitch()` 用户切换字幕时调用

#### 2.7 添加单元测试

**文件**: `test/svc/SettingsService.test.js`

添加 4 个测试用例（SVC-SETTINGS-038 ~ 041）：
- 返回默认播放器配置
- 设置字幕背景颜色
- 部分更新播放器配置保留其他字段
- 播放器配置在 settings 中正确保存
