### 2.2 架构决策

**关键决策：box-view.js 采用独立实现而非继承 box.js**

原有 box-view.html 和 box-view.js 是部分实现，通过加载 box.js 并扩展。此方案存在问题：
- box.js 有自己的 init() 和 DOMContentLoaded 处理器会产生冲突
- box.js 绑定到 box.html 的 DOM 结构，与 box-view.html 不同

**解决方案**：box-view.js 完全独立，包含所有必要逻辑，同时复用 common-utils.js 公共函数。

### 2.3 代码修改

#### 2.3.1 main.js — 添加菜单项和窗口创建

- 在"视图"子菜单添加"收藏记录"项，快捷键 Ctrl+F
- 新增 `createBoxViewWindow()` 函数，创建无边框 BrowserWindow
- 将 `createBoxViewWindow` 传递给 IPC 处理器

```javascript
// 菜单项
{
    label: '收藏记录',
    accelerator: 'CmdOrCtrl+F',
    click: () => { createBoxViewWindow(); }
}

// 窗口创建
function createBoxViewWindow() {
    if (boxViewWindow) { boxViewWindow.focus(); return; }
    boxViewWindow = new BrowserWindow({
        width: 1280, height: 800, minWidth: 800, minHeight: 600,
        title: '收藏记录', frame: false, autoHideMenuBar: true,
        webPreferences: { preload: path.join(__dirname, 'preload.js'), ... },
        show: false
    });
    boxViewWindow.loadFile(path.join(__dirname, 'src', 'renderer', 'box-view.html'));
    boxViewWindow.once('ready-to-show', () => { boxViewWindow.show(); boxViewWindow.maximize(); });
    boxViewWindow.on('closed', () => { boxViewWindow = null; });
}
```

#### 2.3.2 ipc-handlers.js — 添加 IPC 处理器

- 解构新增 `createBoxViewWindow`
- 添加 `open-box-view-window` IPC 处理器

```javascript
ipcMain.handle('open-box-view-window', async () => {
    try {
        createBoxViewWindow();
        return { success: true };
    } catch (error) {
        return { error: error.message };
    }
});
```

#### 2.3.3 preload.js — 暴露 IPC API

