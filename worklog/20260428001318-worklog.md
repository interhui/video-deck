# 演员搜索功能实现工作日志

## 目标

在演员管理页面（actor-mgt.html）中增加演员搜索功能，允许用户通过TMDB搜索演员信息并自动填充到演员表单中。

## 工作过程

### 1. 需求分析

根据需求文档，需要实现以下功能：
1. 在演员模态框的演员名称（actor-name）后增加"搜索"按钮
2. 点击搜索后打开"演员搜索"模态窗
3. 演员搜索模态窗包含：搜索适配器（单选：TMDB演员搜索）、演员名称输入框、搜索按钮
4. 搜索结果以列表形式展示，包含演员ID、名称和照片缩略图，最多显示10个结果
5. 选择演员后获取详细信息并填充到表单，同时下载演员照片

### 2. 代码实现

#### 2.1 修改 actor-mgt.html

**修改点：**
- 在演员名称表单字段后添加"搜索"按钮（id: actor-search-trigger-btn）
- 添加"演员搜索"模态窗（id: actor-search-modal）

**演员搜索模态窗结构：**
- 搜索适配器：单选按钮 TMDB演员搜索
- 演员名称输入框（id: actor-search-input）
- 搜索按钮（id: actor-search-btn）
- 搜索状态区域：加载中（id: actor-search-loading）、错误信息（id: actor-search-error）
- 搜索结果列表容器（id: actor-search-results）
- 确认和取消按钮

#### 2.2 添加 CSS 样式

在 `src/renderer/css/main.css` 中添加演员搜索相关样式：
- `.actor-search-config` - 搜索配置区域
- `.actor-adapter-options` - 适配器选项
- `.actor-adapter-radio` - 单选按钮样式
- `.actor-search-section` - 搜索区域
- `.actor-loading` - 加载状态
- `.actor-error` - 错误信息
- `.actor-results` - 结果列表容器
- `.actor-result-item` - 结果项样式
- `.actor-result-poster` - 演员照片
- `.actor-result-info` - 演员信息
- `.actor-result-name` - 演员名称
- `.actor-result-id` - 演员ID

#### 2.3 添加 IPC 处理器

在 `src/main/ipc-handlers.js` 中添加：
- `tmdb-search-person` 处理器：调用 `tmdbMovieAdapterService.searchPerson(actorName)`
- `tmdb-get-person` 处理器：调用 `tmdbMovieAdapterService.getPerson(personId)`

#### 2.4 添加 Preload API

在 `preload.js` 中添加暴露给渲染进程的 API：
- `tmdbSearchPerson(actorName)` - 搜索演员
- `tmdbGetPerson(personId)` - 获取演员详情

#### 2.5 实现 actor-mgt.js 逻辑

**新增 DOM 元素引用：**
- actorSearchModal, closeActorSearchBtn, actorSearchInput, actorSearchBtn
- actorSearchLoading, actorSearchError, actorSearchResults
- confirmActorSearchBtn, cancelActorSearchBtn, actorSearchTriggerBtn

**新增状态变量：**
- `actorSearchResultsList` - 存储搜索结果
- `selectedActorSearchResult` - 当前选中的搜索结果
- `ACTOR_SEARCH_MAX_RESULTS = 10` - 最大结果数量

**新增函数：**
- `openActorSearchModal()` - 打开演员搜索模态窗
- `resetActorSearchForm()` - 重置搜索表单
- `closeActorSearchModal()` - 关闭演员搜索模态窗
- `searchActorPeople()` - 执行演员搜索
- `renderActorSearchResults(results)` - 渲染搜索结果列表
- `selectActorSearchResult(index)` - 选中搜索结果
- `confirmActorSearch()` - 确认选择演员并填充表单

## 备注

- TMDBAdapterService 中使用 `searchPerson` 和 `getPerson` 方法（而非 `searchActor` 和 `getActor`），已保持一致
- 搜索结果限制为最多10条，由前端常量 `ACTOR_SEARCH_MAX_RESULTS` 控制
- 照片下载复用 `window.electronAPI.downloadMovieCover` 方法
