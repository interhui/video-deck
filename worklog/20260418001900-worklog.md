# 电影海报卡片和电影列表懒加载实现日志

## 日期
2026-04-18

## 实现内容

### 1. 服务层分页支持

#### MovieCacheService.js
- 添加了 `getMoviesPaginated(page, pageSize, filters)` 方法
- 支持分类、标签、评分、演员筛选
- 支持排序
- 返回分页结果 { movies, total, page, pageSize, totalPages }

#### MovieService.js
- 添加了 `getMoviesPaginated(moviesDir, options)` 方法 - 从缓存获取分页数据
- 添加了 `getMoviesPaginatedFromIndex(moviesDir, options)` 方法 - 从index.json快速加载

#### IndexService.js
- 添加了 `getMoviesFromIndexPaginated(category, moviesDir, options)` 方法
- 添加了内部 `sortMovies()` 方法用于排序

### 2. IPC 处理器

#### ipc-handlers.js
- 添加了 `get-movies-paginated` 处理器
- 添加了 `get-movies-paginated-from-index` 处理器
- 添加了 `get-movies-by-category-paginated` 处理器

### 3. preload.js
- 暴露了 3 个新 API:
  - `getMoviesPaginated(params)`
  - `getMoviesPaginatedFromIndex(params)`
  - `getMoviesByCategoryPaginated(params)`

### 4. LazyLoader.js
创建了 `src/renderer/js/utils/LazyLoader.js`，统一管理懒加载逻辑：
- `loadFirstPage()` - 加载第一页
- `loadNextPage()` - 加载下一页
- `handleScroll()` - 检测滚动到底部
- `bindScrollEvent()` / `unbindScrollEvent()` - 绑定/解绑滚动事件
- `destroy()` - 销毁实例
- `reset()` - 重置状态
- `getLoadedCount()` / `getIsLoading()` / `getHasMore()` - 获取状态

### 5. 前端懒加载实现

#### main.js (首页)
- 添加 `lazyLoader` 到 state
- 添加 `initLazyLoader()` 初始化懒加载管理器
- 添加 `loadMoviesAll()` 用于搜索模式的一次性加载
- 修改 `loadMovies()` 使用懒加载（非搜索模式）或一次性加载（搜索模式）
- 修改 `renderMovies(movies, isAppend)` 支持追加模式

#### box.js (电影收藏夹)
- 添加 `lazyLoader` 到 state
- 添加 `initBoxLazyLoader(boxMovieIds, boxMovieMap)` 初始化懒加载管理器
- 添加 `loadMoviesFromBoxAll()` 用于筛选模式的一次性加载
- 修改 `loadMoviesFromBox()` 使用懒加载或一次性加载
- 修改 `renderMovies(movies, isAppend)` 支持追加模式

#### index.html / box.html
- 添加了 LazyLoader.js 脚本引用
