## 1. 任务目标

为电影管理程序增加横版海报展示功能,具体需求如下:

1. 在Setting的"布局设置"中增加"海报样式"(posterStyle),取值为:竖版海报和横版海报,横版海报比例为3:2,包括大、中、小三种海报尺寸
2. 选择横版海报后,首页(index.html)和电影收藏夹(box.html)按照3:2的海报尺寸进行展示
3. 选择横版海报后,详情页(detail.html)中采用上下结构布局,海报部分按照横版海报比例展示,高度最大为300px

## 7. Bug修复记录

### Bug #1: 海报尺寸修改无效
**问题描述**: 选择横版海报后,在Setting修改海报尺寸,海报在index和box中展示没有发生变化。

**根本原因**: CSS使用了`--poster-horizontal-min-width`和`--poster-horizontal-max-width`变量,但JavaScript中更新的是`--poster-min-width`和`--poster-max-width`变量,变量名不一致导致CSS无法读取更新后的值。

**修复方案**: 统一CSS变量命名,`.movies-grid.horizontal-style`使用`--poster-min-width`和`--poster-max-width`变量,与JavaScript保持一致。

**修复文件**: `src/renderer/css/main.css:431-437`

**修复代码**:
```css
.movies-grid.horizontal-style {
    grid-template-columns: repeat(auto-fill, minmax(var(--poster-min-width), 1fr));
}

.movies-grid.horizontal-style .movie-card {
    max-width: var(--poster-max-width);
}
```

**附加修复**: 在`applyLayoutSettings`函数中,当切换到列表视图时移除`horizontal-style`类,避免样式冲突。

**验证结果**: 修复后,修改海报尺寸会立即生效,横版海报的大中小尺寸都能正确展示。
