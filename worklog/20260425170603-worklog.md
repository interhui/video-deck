# 工作日志 - 构建TMDB电影适配服务

## 任务目标
构建TMDBMovieAdapterService服务，用于通过电影名称获取电影详细信息，实现与TMDB API的集成。

## 执行过程

### 1. 需求分析
根据任务要求，需要创建TMDB电影适配服务，包含以下功能：
- **searchMovie**: 根据电影关键字搜索电影列表
- **getMovie**: 根据电影查询ID获取电影详细信息
- **getPerson**: 根据演员ID查询演员详细信息

API集成要求：
- 每次调用TMDB API需要在HTTP Header中增加认证凭证：`Authorization: Bearer <Token>`
- TMDB配置参数包括：url（默认api.themoviedb.org）、token（默认为空）、language（默认zh-CN）

### 2. 代码结构分析
通过探索现有代码库，了解到：
- 服务层架构位于 `src/main/services/` 目录
- 配置服务通过 `SettingsService` 管理
- 默认配置通过 `HardCodeService` 定义
- 单元测试位于 `test/svc/` 目录

### 3. 实现步骤

#### 3.1 创建TMDBMovieAdapterService服务
**文件**: `src/main/services/TMDBMovieAdapterService.js`

**核心功能实现**:

1. **构造函数与配置获取**
   - 接收 `SettingsService` 实例
   - 提供 `getTmdbConfig()` 方法获取TMDB配置（url、token、language）

2. **辅助方法**
   - `buildImageUrl(path, size)`: 构建图片URL，支持w200缩略图和original原图
   - `makeRequest(url, token)`: 封装HTTPS请求，自动添加Authorization header

3. **searchMovie(keyword)** - 电影搜索
   - 参数验证：关键字不能为空
   - Token验证：必须配置TMDB token
   - API调用：`GET /3/search/movie?query=${keyword}`
   - 响应转换：
     - title: 电影名称
     - overview: 电影描述
     - year: 发行年份（从release_date提取）
     - search_id: 电影查询ID
     - poster_url: W200缩略图URL

4. **getMovie(searchId)** - 电影详情获取
   - 参数验证：searchId不能为空
   - Token验证：必须配置TMDB token
   - API调用：`GET /3/movie/${searchId}?append_to_response=credits`
   - 响应转换：
     - movie_id: IMDB ID
     - title: 电影名称
     - overview: 电影介绍
     - genres: 电影标签数组
     - production_companies: 发行公司数组
     - runtime: 电影时长
     - poster_url: 原始尺寸海报URL
     - cast: 演员列表（person_id, name, character, profile_url）
     - directors: 导演列表（person_id, name, profile_url）
     - year: 发行年份

5. **getPerson(personId)** - 演员信息获取
   - 参数验证：personId不能为空
   - Token验证：必须配置TMDB token
   - API调用：`GET /3/person/${personId}`
   - 响应转换：
     - name: 演员姓名
     - birthday: 演员生日
     - biography: 演员备注（介绍）
     - profile_url: 原始尺寸演员照片URL
