# 播放器字幕加载功能开发工作日志

## 任务目标

为播放器实现字幕加载能力，支持SRT和ASS格式的字幕文件，提供自动加载和手动切换两种方式。

## 需求详情

### 功能要求

1. **字幕格式支持**：支持SRT和ASS两种常见字幕格式
2. **自动加载**：
   - 视频播放时，自动查找同目录下的字幕文件
   - 优先级：同名.ass > 同名.srt > 其他.ass（按文件名排序） > 其他.srt（按文件名排序）
   - 例如：视频文件为`珍珠港.mp4`，优先加载`珍珠港.ass`或`珍珠港.srt`
3. **手动切换**：
   - 在播放控制区域增加"字幕切换"按钮
   - 在播放列表右键菜单增加"字幕切换"菜单项
   - 打开字幕切换模态窗，列出同目录下所有字幕文件供选择
4. **字幕显示**：根据时间轴和字幕设置样式在视频播放窗口显示字幕

### 技术要求

1. 字幕加载逻辑代码放在PlayerService.js中
2. 每个新增的后台业务代码都必须生成单元测试并执行通过
3. 代码满足CleanCode要求，保证代码整洁、可读

## 执行过程

### 1. PlayerService字幕功能实现

**文件**：`src/main/services/PlayerService.js`

**新增方法**：

1. `findSubtitleFiles(videoPath)` - 查找视频文件同目录下的字幕文件
   - 扫描视频文件所在目录
   - 过滤出.srt和.ass格式的文件
   - 按文件名排序返回列表

2. `getAutoSubtitle(videoPath)` - 自动加载字幕文件
   - 实现优先级逻辑：同名ASS > 同名SRT > 其他ASS > 其他SRT
   - 返回最匹配的字幕文件信息

3. `parseSRT(subtitlePath)` - 解析SRT格式字幕
   - 读取SRT文件内容
   - 解析时间戳和字幕文本
   - 返回结构化的字幕数据数组

4. `parseASS(subtitlePath)` - 解析ASS格式字幕
   - 读取ASS文件内容
   - 解析Dialogue行
   - 提取时间戳和文本（去除样式标签）
   - 返回结构化的字幕数据数组

5. `parseASSTime(timeStr)` - 解析ASS时间格式
   - 将ASS特有的H:MM:SS.CC格式转换为秒数

6. `loadSubtitle(subtitlePath)` - 加载字幕文件
   - 根据文件扩展名调用对应的解析器
   - 统一的字幕加载入口

### 3. IPC通信实现

**文件修改**：

#### `src/main/ipc-handlers.js`

添加三个IPC处理函数：

```javascript
ipcMain.handle('find-subtitle-files', async (event, videoPath) => {
    try {
        const subtitleFiles = await playerService.findSubtitleFiles(videoPath);
        return subtitleFiles;
    } catch (error) {
        console.error('Error finding subtitle files:', error);
        return [];
    }
});

ipcMain.handle('get-auto-subtitle', async (event, videoPath) => {
    try {
        const subtitle = await playerService.getAutoSubtitle(videoPath);
        return subtitle;
    } catch (error) {
        console.error('Error getting auto subtitle:', error);
        return null;
    }
});

ipcMain.handle('load-subtitle', async (event, subtitlePath) => {
    try {
        const subtitles = await playerService.loadSubtitle(subtitlePath);
        return subtitles;
    } catch (error) {
        console.error('Error loading subtitle:', error);
        return [];
    }
});
```

#### `preload.js`

在electronAPI中暴露字幕相关方法：

```javascript
findSubtitleFiles: (videoPath) => ipcRenderer.invoke('find-subtitle-files', videoPath),
getAutoSubtitle: (videoPath) => ipcRenderer.invoke('get-auto-subtitle', videoPath),
loadSubtitle: (subtitlePath) => ipcRenderer.invoke('load-subtitle', subtitlePath)
```
