# 工作日志 - 电影批量搜索功能实现

## 任务目标

实现电影批量搜索（刮削）功能，允许用户在首页勾选多部电影后，批量从TMDB或R18数据库搜索电影信息并保存到movie.nfo文件。

## 任务详情

### 1. 功能需求
- 在首页(index.html)中，勾选多部电影后，出现"批量搜索"按钮
- 点击按钮后，弹出"批量电影搜索"模态框，包括：工具栏、电影搜索适配器、电影列表
- 工具栏显示搜索进度，包含"搜索"和"取消"按钮
- 电影搜索适配器包含："TMDB电影搜索"和"R18电影搜索"两个选项
- 电影列表为表格样式，包含：电影ID、电影名称、演员、发行商、年份、海报、状态
- 点击"搜索"按钮后，根据选择的适配器逐一搜索电影
- 搜索过程中显示进度，状态分别为："未搜索"、"搜索中"、"完成"、"无记录"
- 搜索完成后，点击"确定"按钮保存搜索内容到movie.nfo文件，海报保存到电影文件夹

### 注意事项
- TMDB搜索需要配置TMDB API Token
- R18搜索需要配置PostgreSQL数据库连接
- 如果搜索无结果，状态显示为"无记录"
- 搜索过程中无法关闭模态框和点击"确定"按钮
