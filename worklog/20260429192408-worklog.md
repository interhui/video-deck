# 批量播放功能实现工作日志

## 任务目标

支持批量播放能力，实现以下三个功能点：
1. 在播放器（player.html）中，playlist-item 增加移除标记，点击移除标记后，将所选的视频从播放列表中移除。如果视频正在播放则停止播放。
2. 在电影首页，运行播放多个勾选后的电影，在"批量删除"后增加"播放"按钮，点击"播放"按钮，将所勾选的电影视频文件添加到player中的播放列表中进行播放
3. 在电影盒子，"导出"按钮后增加"播放"按钮，点击"播放"按钮，将电影盒子中的电影视频文件添加到player中的播放列表中进行播放

### 1. 播放列表移除功能实现

**实现内容：**

#### player.js 修改

在播放器 JavaScript 中添加了 `removePlaylistItem` 函数，用于从播放列表中移除指定索引的视频项。该函数处理以下场景：
- 移除后播放列表为空时，停止播放并清空视频源
- 移除正在播放的视频时，自动播放下一首（或最后一首）
- 移除当前播放之前的视频时，更新当前索引
- 移除其他视频时，仅更新播放列表显示

```javascript
function removePlaylistItem(index) {
    if (index < 0 || index >= playlist.length) return;

    const wasPlaying = index === currentIndex && isPlaying;
    playlist.splice(index, 1);

    if (playlist.length === 0) {
        // 播放列表清空
        elements.videoPlayer.pause();
        elements.videoPlayer.src = '';
        isPlaying = false;
        currentIndex = 0;
        updatePlayPauseBtn();
        elements.playerTitle.textContent = '电影播放';
    } else {
        if (index === currentIndex) {
            // 正在播放的视频被移除
            if (wasPlaying) {
                if (currentIndex >= playlist.length) {
                    currentIndex = playlist.length - 1;
                }
                playItem(currentIndex);
            } else {
                // 停止状态更新显示
                if (currentIndex >= playlist.length) {
                    currentIndex = playlist.length - 1;
                }
                renderPlaylist();
            }
        } else if (index < currentIndex) {
            // 移除的是当前播放之前的项，需要调整索引
            currentIndex--;
            renderPlaylist();
        } else {
            // 移除的是后面的项，只更新显示
            renderPlaylist();
        }
    }
}
```

添加点击事件处理器，收集选中电影的视频文件并打开播放器：
```javascript
elements.batchPlayBtn.addEventListener('click', async () => {
    if (state.selectedMovies.size === 0) {
        alert('请先选择要播放的电影');
        return;
    }

    try {
        const playlist = [];
        const selectedMovieIds = Array.from(state.selectedMovies);

        for (const movieId of selectedMovieIds) {
            const movie = state.movies.find(m => m.id === movieId);
            if (!movie) continue;

            const movieDetail = await window.electronAPI.getMovieDetail(movieId);
            if (!movieDetail || movieDetail.error) continue;

            // 从 fileset 中提取 Main 类型文件
            if (movieDetail.fileset && Array.isArray(movieDetail.fileset)) {
                for (const file of movieDetail.fileset) {
                    const fileType = file.type || file.fileType;
                    if (fileType === 'Main' && file.fullpath) {
                        playlist.push({
                            path: file.fullpath,
                            title: `${movieDetail.name} - ${file.filename || path.basename(file.fullpath)}`,
                            codec: file.codec || file.videoCodec || '',
                            resolution: file.resolution || (file.videoWidth ? `${file.videoWidth}x${file.videoHeight}` : '')
                        });
                    }
                }
            }

            // 备用：使用 original_filename
            if (playlist.length === 0 && movieDetail.original_filename) {
                playlist.push({
                    path: movieDetail.original_filename,
                    title: movieDetail.name || path.basename(movieDetail.original_filename),
                    codec: movieDetail.videoCodec || '',
                    resolution: movieDetail.videoWidth ? `${movieDetail.videoWidth}x${movieDetail.videoHeight}` : ''
                });
            }
        }

        if (playlist.length === 0) {
            alert('所选电影中没有可播放的视频文件');
            return;
        }

        await window.electronAPI.openBatchPlayerWindow(playlist);
    } catch (error) {
        console.error('Error batch playing movies:', error);
        alert('播放失败: ' + error.message);
    }
});
```

## 技术细节说明

### 播放列表项移除逻辑

移除播放列表项时需要处理多种场景：

1. **播放列表清空**：当移除后播放列表为空时，需要：
   - 停止当前播放
   - 清空视频源
   - 重置播放状态
   - 重置标题显示

2. **移除正在播放的视频**：
   - 如果正在播放，自动切换到下一个视频（如果存在）
   - 如果移除的是最后一项，切换到新的最后一项
   - 如果播放列表只剩一项，停止播放

3. **移除当前播放之前的项**：
   - 需要调整 currentIndex（减1）
   - 重新渲染播放列表

4. **移除当前播放之后的项**：
   - 只更新播放列表显示，不影响当前播放

### 批量播放视频收集逻辑

从电影数据中收集视频文件的逻辑：
1. 优先从 `fileset` 数组中提取 `Main` 类型的文件
2. 如果没有 `fileset` 或没有 `Main` 类型文件，使用 `original_filename` 作为备用
3. 为每个视频文件添加标题、编码和分辨率信息
4. 如果没有可播放的视频文件，显示提示信息

### IPC通信流程

批量播放功能通过 IPC 通信实现：
```
渲染进程（main.js/box.js）
    ↓ openBatchPlayerWindow(playlistData)
IPC通道（preload.js）
    ↓ ipcRenderer.invoke('open-batch-player-window')
主进程（ipc-handlers.js）
    ↓ PlayerService.openBatchPlayerWindow()
播放器窗口创建
    ↓ createPlayerWindow({playlist, currentIndex, movieTitle})
渲染进程（player.js）
    ↓ onLoadPlayerData 回调接收数据
播放器开始播放
```
