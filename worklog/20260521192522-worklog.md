## 一、任务目标

在标签管理页面（tag-mgt.html）右侧"标签详情"区域添加电影列表（movie-grid），实现以下功能：
1. 点击左侧标签列表时，显示该标签关联的电影列表
2. 电影列表包含：电影ID、名称、主演、描述、上映时间、发行商、分类、导演、操作
3. 点击电影行时，打开电影详情界面（detail.html）

### 5.2 关键技术点

1. **数据流向**：
   ```
   用户点击标签 
   -> tag-mgt.js: selectTag(tagId)
   -> IPC: get-movies-by-tag
   -> ipc-handlers.js: 获取所有分类电影
   -> TagService.getMoviesByTagId: 篇选标签电影
   -> 返回给前端渲染
   ```

2. **电影列表布局**：
   - 使用 CSS Grid 实现表格布局
   - 8列固定宽度：80px, 150px, 120px, 200px, 80px, 80px, 80px, 80px

3. **单元测试策略**：
   - 测试正常场景、边界场景、异常场景
   - 每个方法至少4个测试用例
