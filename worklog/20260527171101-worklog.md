## 一、任务目标

在播放器界面中实现播放历史记录功能，包括：
1. 在Player.html中增加播放历史记录功能
2. 在电影播放时记录播放信息（播放时间、电影名称）
3. 播放记录写入config文件夹下history.json中
4. 最多保存1000条播放记录
5. 点击"历史记录"按钮查看播放历史
6. 支持根据电影名称和日期筛选历史记录

## 二、执行过程

### 2.1 创建HistoryService服务

**文件**: `src/main/services/HistoryService.js`

**实现内容**:
- 创建HistoryService类管理播放历史记录
- 实现历史记录的加载、保存、添加、筛选功能
- 支持按日期分组存储历史记录
- 自动裁剪超过1000条的记录
- 提供formatDate和formatTime方法格式化时间

**核心方法**:
- `loadHistory()`: 加载历史记录文件
- `addRecord(movieName)`: 添加播放记录
- `trimHistory()`: 裁剪超过1000条的记录
- `saveHistory()`: 保存历史记录到文件
- `filterHistory(movieName, date)`: 筛选历史记录
- `clearHistory()`: 清空历史记录

**数据格式**:
```json
{
    "history": [
        {
            "date": "2026-05-26",
            "records": [
                {
                    "time": "14:30:00",
                    "movie": "ET外星人"
                }
            ]
        }
    ]
}
```
