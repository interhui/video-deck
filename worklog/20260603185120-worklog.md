# 工作日志 - 历史记录视图功能

## 任务目标
在电影管理程序中增加历史记录视图，支持通过菜单和快捷键打开，展示按日期分组的播放历史记录，支持分类/标签过滤、多选播放/导出、删除记录等功能。

## 执行过程

### 1. 代码分析阶段
- 阅读现有代码结构，理解 box.html/box.js 的设计模式作为参考
- 分析 MovieHistoryService.js 已有的业务逻辑（addRecord, filterHistory, deleteRecord等）
- 分析 ipc-handlers.js 中已有的 play-history 相关 IPC 处理器
- 分析 preload.js 中已有的 API 暴露方式
- 分析 main.js 中窗口创建和菜单管理机制

### 2. 后端服务扩展
**文件**: `src/main/services/MovieHistoryService.js`

新增方法：
- `getHistoryDates()`: 返回所有日期列表，按时间倒序排列，用于左侧时间轴展示
- `deleteRecordsByDateAndMovieIds(date, movieIds)`: 按日期和电影ID批量删除记录，用于批量删除功能

### 3. IPC 处理器扩展
**文件**: `src/main/ipc-handlers.js`

新增 IPC 处理器：
- `get-history-dates`: 获取历史记录日期列表
- `delete-history-records`: 按日期和电影ID批量删除记录
- `open-history-window`: 打开历史记录窗口

同时将 `createHistoryWindow` 添加到服务解构参数中。

### 4. Preload API 扩展
**文件**: `preload.js`

新增 API：
- `getHistoryDates()`: 调用 get-history-dates
- `deleteHistoryRecords(date, movieIds)`: 调用 delete-history-records
- `openHistoryWindow()`: 调用 open-history-window

### 5. 主进程窗口创建
**文件**: `main.js`

新增：
- `createHistoryWindow()` 函数：创建无边框的历史记录窗口，加载 history-view.html
- 视图菜单增加"历史记录"菜单项，快捷键 Ctrl+H
- setupIpcHandlers 调用中增加 createHistoryWindow 参数

### 6. 前端视图开发

**文件**: `src/renderer/history-view.html`
- 参考 box.html 结构
- 顶部工具栏：标题、搜索框、最小化/关闭按钮、批量操作按钮
- 主内容区：左侧时间轴列表 + 右侧电影展示区
- 右侧含分类过滤和标签过滤下拉框

**文件**: `src/renderer/js/history-view.js`
- 状态管理：historyData, selectedDate, searchKeyword, currentCategory, currentTag, selectedMovies等
- loadHistoryData(): 加载历史记录并关联电影详情数据
- renderTimeline(): 渲染左侧时间轴列表，点击跳转到对应日期
- renderHistoryContent(): 按日期分组渲染电影卡片，支持搜索/分类/标签过滤
- 复选框多选 → 显示播放/导出批量操作
- 删除按钮 → 调用 deletePlayHistory 删除单条记录
- 点击卡片 → 打开电影详情
- 播放按钮 → 播放电影
- playSelectedMovies(): 多选播放
- exportSelectedMovies(): 多选导出

**文件**: `src/renderer/css/view.css`
- 历史记录视图专用样式
- 时间轴列表样式
- 日期分区和分割线样式
- 电影卡片样式（参考 box.css 中的 movie-card）
- 复选框和删除按钮样式

### 7. 单元测试
**文件**: `test/svc/MovieHistoryService.test.js`

新增测试用例：
- SVC-HISTORY-017: 获取所有日期列表（按时间倒序）
- SVC-HISTORY-018: 空历史记录返回空数组
- SVC-HISTORY-019: 按日期和电影ID批量删除记录
- SVC-HISTORY-020: 删除所有记录后日期条目自动移除
- SVC-HISTORY-021: 删除不存在的movieId不影响历史
- SVC-HISTORY-022: 删除不存在日期的记录不影响历史
